@title render-jsx
@description Defines the JSX syntax for the Retend web framework.
@preamble Retend JSX syntax examples. Retend uses specific attribute naming conventions and has restrictions on expressions and empty tags.

---

@example 1
@input Create a simple div element.
@output
<div />
@note Retend requires self-closing tags for elements with no children.
